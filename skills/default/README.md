# Default Skills

Place end-user Skills for Local Playground under this directory.
Each Skill must follow agentskills format:

- `<skill-name>/SKILL.md` (required)
- `<skill-name>/scripts/` (optional)
- `<skill-name>/references/` (optional)
- `<skill-name>/assets/` (optional)

Skills in `skills/.dev/` are reserved for development workflows and are not part of workspace default skills discovery.
